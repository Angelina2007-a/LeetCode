# 🚀 Crack the Coding Interview – 500 Problem Repository

Welcome to my personal coding repository! This repository documents my journey of solving **500 coding interview problems** with a focus on understanding the thought process behind each solution, not just writing code.

## 👨‍💻 About

This repository contains solutions to coding problems from LeetCode and other interview preparation resources. Each problem is accompanied by detailed notes to make revision easier before coding assessments and technical interviews.

## 📌 What You'll Find

For every problem, I document:

- 📝 Problem statement in my own words
- 💡 Approach & intuition
- 🐍 Python solution
- ⏱️ Time and space complexity analysis
- ⚠️ Edge cases considered
- 🔧 Mistakes encountered and how they were fixed
- 🎯 Key takeaways and patterns to remember
